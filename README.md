# An analytic theory of creativity in convolutional diffusion models

Hacker presentation for IFT6164 at Mila and UdeM. Based off the paper of the same name (https://arxiv.org/abs/2412.20292).

## Installation

### Prerequisites

- Python 3.8 or higher
- pip

### Install the Package
Install the package in development mode (in root):
```bash
pip install -e .
```
